# Unidirectional-power-analyzer
基于瑞萨的单相功率分析仪 2025年嵌赛国三作品（没国二是因为pcb没加丝印 碎碎念.......）
这里提供核心代码供参考，可以参考阅读作品设计报告，因为考虑到瑞萨的不同单片机功能函数略有差异，因此不上传完整工程。需要的同学参考核心代码进行配置即可

## 项目简介

本项目为基于瑞萨 Renesas 单片机平台设计的单相功率分析仪，是 2025 年嵌入式芯片与系统设计竞赛作品。项目面向单相交流电参数采集与分析场景，主要实现电压、电流信号采样、有效值计算、功率参数计算、频率检测、功率因数分析以及结果显示等功能。

仓库当前仅提供项目核心代码和关键实现思路，并未上传完整瑞萨工程。原因是不同型号的瑞萨单片机在 FSP 配置、外设初始化、ADC 通道、定时器、DMA、串口、显示接口等方面存在差异，直接上传完整工程不一定能在其他开发板上无修改运行。因此，本仓库更适合作为算法逻辑、功能模块和工程移植参考。

## 项目完成内容

本项目目前已经完成以下核心功能：

1. 单相电压、电流信号采集  
   通过 ADC 对电压采样通道和电流采样通道进行周期性采样，为后续参数计算提供原始数据。

2. 电压、电流有效值计算  
   对采样数据进行偏置校正、平方平均和开方处理，计算交流电压有效值和交流电流有效值。

3. 有功功率计算  
   根据瞬时电压和瞬时电流的乘积计算瞬时功率，并对一个或多个周期内的功率数据求平均，得到有功功率。

4. 视在功率与功率因数计算  
   根据电压有效值和电流有效值计算视在功率，并结合有功功率计算功率因数。

5. 频率检测  
   通过采样波形的过零点、周期估计或定时器计数方式，计算输入交流信号频率。

6. 数据显示与交互  
   将电压、电流、功率、频率、功率因数等结果输出到显示模块或串口终端，方便用户观察实时测量结果。

7. 工程核心代码整理  
   仓库保留了核心算法与主要功能代码，便于其他瑞萨开发板或类似单片机平台进行移植。

主要功能模块说明
1. ADC 采样模块

ADC 采样模块负责采集电压和电流输入信号。通常需要配置两个 ADC 通道：

电压采样通道：用于采集电压传感器或分压电路输出
电流采样通道：用于采集电流传感器或采样电阻调理后的输出

该模块通常包含以下功能：

ADC 初始化
ADC 启动采样
ADC 采样完成回调
电压 / 电流原始数据读取
采样数组缓存

移植时需要重点确认：

ADC 分辨率
ADC 参考电压
ADC 通道编号
采样频率
触发源
是否使用 DMA
采样数组长度
2. 定时器模块

定时器模块通常用于产生稳定的采样周期，使 ADC 按固定频率采集数据。

该模块通常完成：

定时器初始化
定时中断配置
ADC 触发
采样周期控制
频率测量辅助

移植时需要注意：

定时器时钟源
计数周期
中断优先级
ADC 触发方式
目标采样频率是否满足计算要求

对于 50 Hz 或 60 Hz 单相交流信号，采样频率不能过低。一般需要保证每个周期内有足够多的采样点，否则有效值、功率和频率计算都会不稳定。

3. 数据预处理模块

由于 ADC 采集到的是经过偏置后的模拟信号，不能直接作为交流瞬时值使用。一般需要先做预处理。

常见处理包括：

去直流偏置
零点校准
比例系数换算
简单滤波
异常值剔除

例如，ADC 采样值需要先减去中点偏置，再根据传感器比例系数换算成真实电压或电流。

4. 有效值计算模块

有效值计算一般采用 RMS 方法：

Vrms = sqrt((v1² + v2² + ... + vn²) / n)
Irms = sqrt((i1² + i2² + ... + in²) / n)

该模块用于计算：

电压有效值
电流有效值

计算时需要注意：

必须先去除直流偏置
采样窗口最好覆盖完整周期
采样点数过少会导致结果波动
比例系数需要根据实际硬件标定
5. 功率计算模块

功率计算通常包括瞬时功率、有功功率、视在功率和功率因数。

瞬时功率：

p(t) = u(t) × i(t)

有功功率：

P = average(p(t))

视在功率：

S = Vrms × Irms

功率因数：

PF = P / S

该模块需要保证电压和电流采样点在时间上对应。如果电压、电流采样存在明显延迟，功率因数和有功功率会出现误差。

6. 频率检测模块

频率检测可通过过零检测、峰值间隔或周期估计实现。常见做法是检测电压波形的连续过零点，根据两个相邻同方向过零点之间的时间计算周期，再换算频率。

需要注意：

过零点附近容易受噪声影响
需要设置阈值或滞回判断
采样频率越高，频率计算越精确
输入信号畸变会影响过零检测稳定性
7. 显示与通信模块

项目可通过 LCD、OLED、串口或上位机显示计算结果。

常见显示内容包括：

电压有效值 Vrms
电流有效值 Irms
有功功率 P
视在功率 S
功率因数 PF
频率 Freq

如果使用串口输出，建议输出格式保持稳定，方便后续上位机解析。例如：

Vrms: 220.15 V
Irms: 0.52 A
P: 105.30 W
S: 114.48 VA
PF: 0.92
Freq: 50.01 Hz

移植说明

本项目不是完整可直接编译工程，移植时不能直接复制文件后编译运行。需要根据目标瑞萨芯片和开发板重新配置工程。

1. 新建瑞萨 FSP 工程

建议先在 e² studio 或 Renesas FSP 环境中新建一个与你目标开发板匹配的工程。

需要确认：

MCU 型号
开发板型号
FSP 版本
编译器版本
调试器型号
时钟配置
引脚配置

不要直接假设不同瑞萨芯片的 ADC、定时器、串口和 DMA API 完全一致。

2. 重新配置 ADC

移植时必须根据实际硬件修改 ADC 配置。

重点确认：

电压采样 ADC 通道
电流采样 ADC 通道
ADC 参考电压
ADC 分辨率
采样时间
触发方式
是否连续采样
是否使用扫描模式
是否使用 DMA

如果 ADC 通道映射错误，程序可能能运行，但测量数据完全不对。

3. 重新配置定时器

定时器决定采样频率。采样频率不稳定会直接影响有效值、功率和频率计算。

需要确认：

定时器周期
定时器中断
ADC 触发源
采样频率
中断优先级

如果使用 50 Hz / 60 Hz 工频测量，建议保证每个周期有足够采样点。

4. 修改标定参数

不同硬件的传感器、电阻分压、电流采样电路和运放增益都不同，因此必须重新标定。

需要修改：

电压比例系数
电流比例系数
ADC 零点偏置
传感器偏移量
滤波参数

不要直接使用原工程中的比例系数，否则显示数值可能与实际电压电流不一致。

5. 注意电压电流相位误差

如果电压通道和电流通道存在采样时间差、传感器延迟或滤波相移，会影响功率因数和有功功率计算。

移植时建议：

尽量同步采样电压和电流
减少两个通道的采样延迟
必要时做相位补偿
用标准负载校准功率因数
6. 注意安全问题

本项目涉及交流电测量。移植和测试时必须注意电气安全。

建议：

使用隔离电压传感器或隔离采样方案
调试阶段使用低压交流信号源
不要直接将市电接入单片机 ADC
保证模拟前端不会超过 ADC 输入范围
检查 PCB 安规间距和接地设计

单片机 ADC 通常只能接受 0 到参考电压范围内的信号，输入过压可能直接损坏芯片。

7. 注意完整工程缺失

当前仓库只提供核心代码参考，不包含完整可直接编译的瑞萨工程。移植时需要自行补充：

推荐调试流程

建议按照以下顺序调试：

1. 先点亮基础工程，确认工程可编译、可下载、可运行
2. 调通串口输出，方便打印调试信息
3. 单独测试 ADC 原始采样值

5. 输入低压交流信号，检查波形采样是否正常
6. 调试 Vrms / Irms 计算
7. 调试频率检测
8. 接入电压电流传感器，重新标定比例系数
9. 使用标准负载校准功率和功率因数
10. 最后接入显示模块或上位机显示
